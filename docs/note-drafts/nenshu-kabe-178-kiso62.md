## はじめに

「2026年に103万円の壁が変わったと聞いたけど、いくらになったの？」
「基礎控除が上がると手取りはどのくらい増える？」

令和8年（2026年）の税制改正で、給与所得控除と基礎控除が引き上げられ、いわゆる「年収の壁」が大幅に変わりました。この記事では、変更の内容と手取りへの影響を、公式情報をもとに解説します。

※本記事の計算例はすべて概算です。実際の税額・手取りは年収・家族構成・他の控除等により異なります。

---

## 何が変わったのか（令和8年の改正）

令和7年度税制改正により、令和8年・9年（2026〜2027年）の2年間を対象とした特例措置として、以下が変更されました。

### ①給与所得控除の最低保障額の引き上げ

| | 改正前（〜令和7年） | 改正後（令和8・9年） |
|---|---|---|
| 給与所得控除の最低保障 | 55万円 | **74万円** |

給与収入が低い方ほど恩恵が大きく、年収220万円以下の方は控除額が最大**19万円**増えます。

### ②基礎控除の引き上げ（本則62万円＋特例上乗せ）

基礎控除（所得税）は合計所得金額に応じて変わります。

| 合計所得金額 | 改正前（〜令和7年） | 改正後（令和8・9年） |
|---|---|---|
| 489万円以下 | 48万円 | **104万円**（本則62万＋特例42万） |
| 655万円以下 | 48万円 | **67万円**（本則62万＋特例5万） |
| 2,350万円以下 | 48万円 | **62万円**（本則のみ） |
| 2,400万円以下 | 32万円 | 48万円 |
| 2,450万円以下 | 16万円 | 32万円 |
| 2,500万円以下 | 8万円 | 16万円 |
| 2,500万円超 | 0円 | 0円 |

「基礎控除62万円」とは、新しい**本則の控除額**のことです（改正前の本則は48万円）。年収が比較的低い方（給与所得489万円以下≒年収約655万円以下）は、特例上乗せによりさらに大きい控除が受けられます。

**出典：** 国税庁「令和7年度税制改正のあらまし」・財務省令和7年度税制改正（大綱）

## 「103万円の壁」が「178万円の壁」に

従来の「103万円の壁」は、給与所得控除（最低保障）＋基礎控除の合計で決まります。

| | 改正前（〜令和7年） | 改正後（令和8・9年） |
|---|---|---|
| 給与所得控除（最低保障） | 55万円 | 74万円 |
| 基礎控除（所得税・合計所得489万円以下） | 48万円 | 104万円 |
| **合計（≒所得税がかからない年収の目安）** | **103万円** | **178万円** |

つまり、年収178万円以下の方（給与のみ・独身・扶養なし・合計所得489万円以下の場合）は、**令和8・9年分は所得税がかからない**目安になります。

ただし、これは**所得税**の話です。社会保険の扶養（106万円・130万円の壁）は別の制度で変わっていません。

## 注意事項

- 令和8・9年の**特例措置**です。令和10年以降の扱いは今後の改正に依ります。
- **住民税の基礎控除は別途**：住民税の基礎控除も改正されましたが、所得税とは控除額が異なります。
- **社会保険の扶養基準は変わっていません**：106万円・130万円の壁（社会保険の被扶養者要件）は今回の税制改正では変更されていません。
- 上記の手取り計算はあくまで概算です。実際は雇用保険料・住民税の均等割・調整控除等も影響します。

## まとめ

- 令和8・9年の特例で基礎控除（本則）が48万→**62万円**に、さらに特例上乗せで最大**104万円**に
- 給与所得控除最低保障も55万→**74万円**に引き上げ
- 「103万円の壁」が**178万円の壁**に変わった（所得税のかからない年収の目安）
- 手取りは年収にもよるが**5〜11万円程度増える**見込み（概算）
- 社会保険の扶養基準（106万円・130万円）はこの改正では変わっていない
