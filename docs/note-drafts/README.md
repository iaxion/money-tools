# note 記事ドラフト — 編集ルールとテンプレート

money-tools の計算ツールと公式データを活用したnote.com記事の原稿置き場。
運営者が `note.com` に貼って公開するための「貼るだけ」完成原稿を管理する。

## ディレクトリ構成

```
docs/note-drafts/
├── README.md        このファイル（テンプレ・ルール）
├── backlog.md       記事候補リスト（優先度付き）
└── *.md             公開原稿（ファイル名 = スラッグ案）
```

## ガードレール（YMYL・厳守）

- **公式値のみ・出典URL明記**：数値は国税庁・厚生労働省・協会けんぽ等の一次情報を確認し、URLを注釈に記載する。
- **「概算」明記**：計算結果はすべて概算であることを本文・末尾に明示する。
- **施行日・対象年度を必ず記載**：改正ネタは「いつから変わるか」「対象は誰か」を冒頭に明記。確定していない情報は「予定」と書く。
- **オリジナル解説のみ**：公式文書のコピペ・無断転載はしない。有用なオリジナル説明を書く。
- **PR/アフィリ明示**：送客リンクがある場合は「PR」または「広告」と明記（景表法・ステマ規制）。
- **スパム投稿しない**：人的ペース（週1〜2本）で公開。量より質。

## 記事テンプレート

```markdown
---
タイトル: 【検索意図語を含む】〇〇について知っておきたいこと
最終確認日: YYYY-MM-DD
対象年度・施行日: 令和◯年（YYYY年）／YYYY年MM月DD日施行（または「予定」）
出典: [機関名](URL)
---

## はじめに（読者の不安 → この記事でわかること）

〇〇が気になっている方も多いのではないでしょうか。
この記事では、〇〇について【公式情報をもとに】わかりやすく解説します。

※本記事の計算例は概算です。実際の金額は個人の状況・自治体・加入保険者により異なります。

## 〇〇とは（本文セクション）

...本文...（公式値のみ・推測で数値を書かない）...

## あなたの数字で試す

▶ [〇〇シミュレーター（無料）](https://money-tools.pages.dev/tools/〇〇/)（PR）

## まとめ

...要点3点以内...

---
**出典・参考**
- [機関名](公式URL)
- [機関名](公式URL)

※本記事は概算の説明を含みます。正確な金額は税理士・加入保険者等にご確認ください。
```

## 公開フロー

1. Agent がこのディレクトリに原稿（.md）を追加・更新 → PR → main にマージ
2. 運営者が note.com にコピー＆ペーストして公開
3. 公開後、URLを Issue またはこのREADMEの「公開済み」欄に記録
4. 改正があれば Agent が原稿を更新 → 運営者が note を編集
